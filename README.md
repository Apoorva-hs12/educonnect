# EduConnect - Luxury Learning Management System

EduConnect is a premium, state-of-the-art educational platform designed to provide a seamless and visually stunning experience for both students and instructors. Built with a modern technical stack, it focuses on high-performance interactions, real-time communication, and a "Luxury Technical" design aesthetic.

## ✨ Key Features

- **Dynamic Dashboards**: Personalized views for students and instructors featuring analytics and progress tracking.
- **Real-time Interaction**: Integrated messaging and live class management using Socket.IO.
- **Course Management**: Full CRUD capabilities for instructors to create, manage, and distribute course content.
- **Interactive Analytics**: Premium data visualization for tracking user performance and engagement.
- **Luxury UI/UX**: Built with Framer Motion for smooth transitions, Glassmorphism effects, and a sleek dark-themed aesthetic.
- **Robust Authentication**: Secure state management using Zustand and protected API routes.

## 🛠️ Tech Stack

### Frontend
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion
- **State Management**: Zustand
- **Icons**: Lucide React
- **Routing**: React Router 7

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Language**: TypeScript
- **Database**: MongoDB (Mongoose)
- **Real-time**: Socket.IO
- **Validation**: Zod
- **Logging**: Winston

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas account or local installation
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Apoorva-hs12/educonnect.git
   cd educonnect
   ```

2. **Backend Setup**:
   ```bash
   cd backend
   npm install
   ```
   - Create a `.env` file in the `backend` directory and add your configurations (refer to `.env.example` if available).
   - Run in development: `npm run dev`

3. **Frontend Setup**:
   ```bash
   cd ../frontend-vite
   npm install
   npm run dev
   ```
